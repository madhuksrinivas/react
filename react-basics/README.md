# React Basics

A React learning project with practical examples and documentation.

## Setup

```bash
npm install
npm run dev
```

## Topics Covered

- **JSX** - JavaScript XML syntax
- **Props** - Passing data between components
- **Render Props** - Sharing code pattern
- **List Rendering** - Rendering arrays with keys
- **Higher Order Components** - Component composition
- **Context API** - Global state management
- **Conditional Rendering** - Display logic
- **Components** - Controlled vs Uncontrolled
- **Hooks** - useState, useEffect, useReducer, useContext, useRef, useMemo, useCallback, custom hooks
- **Component Architecture** - Composition, Container/Presentation, Lifting State, Prop Drilling, Reusable Components
- **Event Handling** - User interactions
- **Error Boundaries** - Error handling
- **Lazy Loading & Suspense** - Code splitting
- **Routing** - Navigation with React Router
- **Forms** - Form handling and validation

## Structure

Each topic has:

- Working examples (`.jsx` files)
- Documentation (`.txt` files)

## Usage

Uncomment components in `src/App.jsx` to view different examples.
