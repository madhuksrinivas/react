# HomePage Component

## Overview

The landing page of the React Query demo application. Provides navigation to different React Query pattern examples.

## Purpose

- Serves as the entry point for the application
- Displays welcome message
- No data fetching or React Query logic

## Usage

```jsx
<HomePage />
```

## Features

- Simple static page
- Navigation handled by React Router in App.jsx

## Related Components

- All other components are accessible from navigation links in App.jsx
