# Getting Started

## Prerequisites

- Node.js 16+
- npm or yarn

## Installation

```bash
# Clone repository
git clone <repo-url>

# Install dependencies
npm install

# Start JSON server (API)
npm run serve-json

# Start development server
npm run dev
```
